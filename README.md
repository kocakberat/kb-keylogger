# kbkey
This is a malware/forensics project, consisting of two phases.

## 1st Phase
I have a leftover microcontroller from a university class (WaveShare ESP32-S3 Mini), which I will use as a keylogger written in C++ for Linux OS Systems. It will log the keypoard inputs and send it periodically somewhere. 

## 2nd Phase
I will forensically analyse one of my own machines, and analyse the network traffic of the machine, and see the keylogging activity.
