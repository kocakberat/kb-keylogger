# kbkey
This is a malware and forensics project, consisting of two phases.

## 1st Phase
I have a leftover microcontroller from a university class (WaveShare ESP32-S3 Mini), which will function as a keylogger written in C++ for Linux/Windows/MacOS. It will capture the keypoard and periodically send the logged data to a remote destination.

## 2nd Phase
I will forensically analyse one of my own machines, and examine the network traffic generated by the keylogger and identify the exfiltration activity. 

For detailed information, see [Writeup](./docs/writeup.typ).
